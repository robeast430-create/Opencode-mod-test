# Opencode Mod Test

A Deltarune mod.

## Mod Structure

```
Opencode-mod-test/
├── carl.json          # Mod info file
├── scripts/           # GML script files
├── sprites/           # Sprite files
├── sounds/            # Sound files
├── rooms/             # Room data files
├── objects/           # Object data files
└── data/              # Additional data files
```

## Installation

1. Install UndertaleModLoader (TML) for Deltarune
2. Place this mod folder in your `mods/` directory
3. Run the game with the mod enabled
